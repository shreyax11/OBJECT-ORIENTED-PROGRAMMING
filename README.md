# Object-Oriented Programming in C++

This repository contains C++ programs based on Object-Oriented Programming (OOP) concepts.
It is created for learning, practice, and academic purposes.

## Concepts Covered
- Classes and Objects
- Constructors and Destructors
- Function Overloading
- Operator Overloading
- Arrays and Strings
- Basic File Handling
- Menu-driven Programs
- Simple Mini Projects

## Programs Included
- Calculator Program
- Bank Management System
- Student Report Card System
- Complex Number Operations (Operator Overloading)
- Array Operations (Reverse, Rotate, Union, Intersection)
- Matrix Operations
- Unit Converter
- Electricity Bill Calculation

## Language Used
- C++ (Standard C++)

## Purpose
This repository is maintained to strengthen OOP concepts in C++
and for academic and practical understanding.
